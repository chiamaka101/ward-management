


ward management project 

 A django web app for hospital to admit patients, track illness,presciption, bill balance and discharge them.

 core features
 Add new patients with name, illness, prescription and bill
 Display patients as card  with dynamic status - red "patients still owes money" if bill is bill >0, green "cleared to go home"if bill =0
 one click discharge to delete patient records

 purpose: Beginner CRUD project covering  model,views,url roouting and django template logic like {% if %} and {% for %}

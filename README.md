# Ward Management

## Overview
This is a comprehensive ward management system designed to streamline the operations and management of healthcare wards. The system includes features for patient tracking, bed management, staff scheduling, and reporting.

## Features
- **Patient Tracking:** Monitor patient admissions, discharges, and transfers.
- **Bed Management:** Keep track of bed availability and allocation.
- **Staff Scheduling:** Plan shifts and manage staff assignments efficiently.
- **Reporting:** Generate reports for various metrics related to ward operations.

## Installation
1. Clone the repository: `git clone https://github.com/chiamaka101/ward-management.git`
2. Navigate into the project directory: `cd ward-management`
3. Install dependencies: `npm install`

## Usage
To start the application, run: `npm start`
